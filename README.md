# 🎨 Atelier — твой AI-стилист

Поздравляю! У тебя на руках готовое React-приложение. Эта инструкция покажет, как его разместить в интернете, чтобы им могли пользоваться другие люди.

**Время на развёртывание: ~1 час, если впервые.**

---

## 📋 Что ты будешь делать

1. Получишь API-ключ от Anthropic (это "пропуск" для общения с Claude)
2. Создашь аккаунт на GitHub (чтобы хранить код)
3. Создашь аккаунт на Vercel (чтобы разместить приложение)
4. Загрузишь код через Vercel — и получишь ссылку, которой можно делиться

---

## 🔑 Шаг 1 — получить ключ Anthropic (15 минут)

1. Зайди на **console.anthropic.com** → нажми **"Sign up"** (если нет аккаунта)
2. Подтверди email, войди
3. В левом меню выбери **"API Keys"** → **"Create Key"**
4. Назови ключ `atelier-stylist`, скопируй его и сохрани в надёжном месте (например, в заметки) — **этот ключ больше нельзя будет посмотреть**, только создать новый
5. В меню **"Plans & Billing"** положи на счёт **$5-10** — этого хватит на ~2000 образов

⚠️ **Важно про безопасность:** никому не показывай этот ключ. Если покажешь — кто-то потратит твои деньги. Ключ должен жить только на сервере Vercel (увидишь дальше, как).

**💰 Сколько это стоит?** Один образ = примерно $0.005 (полкопейки). Если у тебя будет 100 пользователей и каждый сгенерирует 10 образов за месяц = $5 в месяц. Если 1000 пользователей — $50.

---

## 💾 Шаг 2 — создать GitHub-аккаунт (5 минут)

GitHub — это место, где хранится код в интернете.

1. Зайди на **github.com** → нажми **"Sign up"**
2. Укажи email, придумай пароль, подтверди email
3. Всё, аккаунт готов (настройки внешнего вида можешь пропустить)

---

## 📦 Шаг 3 — загрузить код на GitHub (10 минут)

Самый простой способ — через веб-интерфейс GitHub, без скачивания программ.

1. На github.com нажми зелёную кнопку **"New"** (или "+" сверху → "New repository")
2. Назови репозиторий `atelier-stylist`
3. Оставь **"Public"**
4. Поставь галочку **"Add a README file"**
5. Нажми **"Create repository"**

Теперь нужно загрузить файлы приложения.

6. На странице репозитория нажми **"Add file"** → **"Upload files"**
7. Перетащи в окно **все файлы из пакета**:
   - `App.jsx`
   - `package.json`
   - `README.md` (этот файл)
   - Папку `api` с файлом `claude.js` внутри
   - Папку `src` со всем содержимым
   - Файлы `index.html`, `vite.config.js`, `tailwind.config.js`, `postcss.config.js`
8. Внизу нажми **"Commit changes"**

Готово — твой код теперь в интернете!

---

## 🚀 Шаг 4 — развернуть на Vercel (15 минут)

Vercel — это хостинг, который возьмёт твой код и превратит его в сайт.

1. Зайди на **vercel.com** → **"Sign Up"** → выбери **"Continue with GitHub"** (так проще всего)
2. Разреши Vercel доступ к твоему GitHub
3. На главной странице Vercel нажми **"Add New..."** → **"Project"**
4. Увидишь список своих GitHub-репозиториев → найди `atelier-stylist` → нажми **"Import"**
5. На странице настроек проекта:
   - **Framework Preset:** выбери **"Vite"** (должен определиться автоматически)
   - Раскрой блок **"Environment Variables"**
   - Добавь переменную:
     - **Name:** `ANTHROPIC_API_KEY`
     - **Value:** вставь сюда свой ключ из Шага 1 (начинается с `sk-ant-...`)
   - Нажми **"Add"**
6. Нажми большую кнопку **"Deploy"**
7. Подожди 1-2 минуты — Vercel соберёт приложение

Когда появится надпись **"Congratulations!"** — поздравляю! 🎉

Нажми на превью — и ты попадёшь на свой сайт по адресу вида `atelier-stylist.vercel.app`. **Это и есть ссылка, которую можно отправлять подругам!**

## 🎨 Как изменить что-то потом?

Допустим, ты хочешь поменять текст или цвет. Два способа:

**Через веб-интерфейс GitHub (просто):**
1. Зайди на github.com/твой-ник/atelier-stylist
2. Открой файл, который хочешь поменять → нажми иконку карандаша
3. Внеси правки → **"Commit changes"**
4. Vercel автоматически обновит сайт за 1-2 минуты

**С помощью Claude (ещё проще):**
Вернись в этот чат, скажи что хочешь поменять — я подготовлю новую версию файла. Дальше ты просто заменишь его на GitHub.

## 🌐 Хочешь красивый домен?

Адрес `atelier-stylist.vercel.app` — ок, но можно купить свой, например `atelier-stylist.com`.

1. Купи домен на **namecheap.com** или **reg.ru** (~$10-15 в год)
2. На Vercel зайди в **Settings → Domains** твоего проекта
3. Нажми **"Add Domain"**, введи купленный домен
4. Vercel покажет DNS-настройки — скопируй их
5. На сайте, где купила домен, в настройках DNS добавь эти записи
6. Через 1-24 часа домен заработает

## ⚠️ Что важно помнить

**Безопасность ключа:**
Твой API-ключ Anthropic хранится только на сервере Vercel в Environment Variables. Он никогда не попадает в код и не виден пользователям. Это безопасно.

**Контроль расходов:**
Зайди на console.anthropic.com → **"Plans & Billing"** → установи **лимит расходов** (Monthly spending limit). Например, $50 в месяц. Если лимит исчерпан — API просто перестанет отвечать, не потратит больше.

**Данные пользователей:**
Сейчас профили, гардеробы и референсы сохраняются в браузере каждого пользователя (в `localStorage`). Это значит, что если пользователь очистит кэш или сменит устройство — данные пропадут. Для продакшена на большую аудиторию это нужно заменить на настоящую базу данных — но для проверки идеи сойдёт.

**Юридические моменты:**
Если приложение собирает хоть какие-то данные (в твоём случае — да, пусть и в браузере), формально нужна политика конфиденциальности. Для дружеского прототипа — не критично, для серьёзного запуска — обязательно.

## 🆘 Что-то пошло не так?

- **Vercel показывает ошибку сборки** — скорее всего, забыли один из файлов. Проверь, что в репозитории есть ВСЕ файлы из пакета
- **Сайт открывается, но образы не генерируются** — проверь Environment Variable `ANTHROPIC_API_KEY` в настройках Vercel. Он должен начинаться с `sk-ant-`
- **Ключ закончился / нет денег** — положи денег на console.anthropic.com в разделе Billing
- **Что-то другое** — возвращайся в этот чат, описывай что видишь, разберёмся
